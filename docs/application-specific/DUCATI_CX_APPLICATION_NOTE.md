# Original Ducati-Oriented Application Context

This project was originally created as a Ducati-oriented customer-experience analytics portfolio case study.

The purpose was to show how I would structure:

- CX signal collection;
- customer segmentation;
- journey-stage reporting;
- stakeholder-ready dashboards;
- insight-to-action tracking;
- privacy-aware communication of findings.

The project uses simulated data only. It does not claim access to Ducati systems, Ducati customer data, dealer data, telemetry, dashboards, internal processes or production tools.

It should be presented as an analytics and communication portfolio project, not as a production system or an affiliated company project.
